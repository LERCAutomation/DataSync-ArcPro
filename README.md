Data Extractor
==============

DataExtractor tool for ArcGIS Pro

Copyright © 2024 Andy Foy Consulting

Overview
--------
DataExtractor is an ArcGIS Pro tool and associated SQL scripts to extract data from SQL Server and ArcGIS Pro GIS layers for pre-defined spatial areas. The output files can be created as ESRI Geodatabase (.gdb) files, ESRI Shapefiles (.shp),  files and/or Comma Separated Value (.csv) files.

Requirements
------------
The DataExtractor tool requires the following:

 - ArcGIS Pro 3.3 or higher

 - Microsoft SQL Server 2008 Express Edition or later, OR
 - Microsoft SQL Server 2008 or later

Other versions of the tool are available for:

 - ArcGIS 10.1 or higher
 - MapInfo 32bit or 64bit
